# Full Refund

> Return the full order total amount.

**Step 1.** Choose a **Fully Captured**/**Partially Captured** order from the **All Orders** list to be refunded, to go into its Order Details page then click on the `Refund` button.

**Step 2.** Enter the total amount to be refunded and a related refund comment for validating the refund, then click on the `Submit` button.

**Step 3.** Confirm the refund request by clicking on the `Refund Order` button.
