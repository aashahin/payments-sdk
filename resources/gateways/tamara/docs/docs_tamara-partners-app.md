# Refunding Orders

**Tamara** Partners app offers a seamless refund process

1. Click on **Refund an order** button
2. Enter *order reference ID*(e.g. Transaction ID, Receipt Number, Order Number, Order ID...etc)
3. Confirm this is the correct order details that you wish to refund
4. Enter amount to be refunded to customer up to 100% of the total amount of the order and enter a comment for reference.
5. Amount will be refunded to the customer's payment method
